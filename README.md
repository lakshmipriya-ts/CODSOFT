# CODSOFT — Internship Tasks

This repository contains internship tasks for Lakshmipriya TS. The tasks are organized under the `tasks/` folder so each assignment is easy to find and review.

Structure

- tasks/level-1-portfolio — Level 1: Personal portfolio (HTML & CSS)
- tasks/level-2-landing — Level 2: Landing page (HTML & CSS)
- tasks/level-3-calculator — Level 3: Calculator (HTML, CSS, JS)

How to view

Open the files in each folder (index.html) in your browser. Replace any placeholder images and the `resume.pdf` with your real files.
